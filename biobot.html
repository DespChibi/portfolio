Demo Application BioBot

This was a project I started in July 2020 when I first learned JavaScript. At first it was made for a university project. The teacher really liked the idea and became a co-creator.

We developed the app and presented it to the Aegean University conference of Modern Society, Education and Mental Health in 2020.

The app is a chat bot that allows the user to retrieve biological information.

The user has the ability to pronounce a biological term and the chat bot returns its definition both in auditory an written form.

This application is a demo app and it may be fully developed in the future.

Technologies:
  • - JavaScript
  • - Html/CSS
  • - Google Speech API
