Demo Application VR-SONIC

This was a project I started in May 2021 as a university project. The teacher really liked the idea and became a co-creator.

We developed the app and presented it to the Avarts University conference of DCAC in 2021.

The app is a virtual reality environment that allows the user to navigate it using their voice.

The user has the ability to pronounce a navigation command and the application updates their position.

This application is a demo app and it may be fully developed in the future.

Technologies:
  • - A-Frame
  • - Html
  • - Google Speech API
